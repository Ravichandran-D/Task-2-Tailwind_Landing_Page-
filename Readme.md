# HTML Landing Page with Tailwind CSS

This project is a simple HTML landing page styled using [Tailwind CSS](https://tailwindcss.com/).

## Features

- Responsive landing page layout
- Modern design with Tailwind CSS utility classes
- Easy to customize and extend

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) and npm (optional, for using Tailwind CLI)
- Basic knowledge of HTML and CSS

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Ravichandran-D/Task-2-Tailwind_Landing_Page-.git
    cd Task 2(Tailwind_Landing_Page)
    ```
2. Install Tailwind CSS (optional, if you want to customize styles):
    ```bash
    npm install -D tailwindcss
    npx tailwindcss init
    ```
3. Alternatively, you can use the CDN in your HTML file:
    ```html
    <script src="https://cdn.tailwindcss.com"></script>
    ```

### Usage

1. Open `index.html` in your browser to view the landing page.
2. Edit the HTML and Tailwind classes to customize the design.
